# 8020fit - Pace University Capstone Project

***

## Project Description

8020fit is a comprehensive web app designed to optimize nutrition and fitness tracking. End users will take in depth quiz on what the nutrition and fitness goals are as well as lifestyle habits as well as nutrition/food preferences. From there, they will put in their body measurements and goal body measurements and any other information. The AI will then create recommendations for you for diet and exercise. You will also be able to track weights added to workout, macros, etc. which will then create stunning visuals to give the end user a macro of their nutrition and fitness to see if theyâ€™re on track.
